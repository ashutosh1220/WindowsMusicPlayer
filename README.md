# WindowsMusicPlayer
# 🎵 Music Player

A modern desktop music player built with **WinUI 3** and the **Windows App SDK**, designed to deliver a native Windows 11 experience with a highly polished UI. Every control is custom-painted, every interaction is intentional.

---

## ✨ Features

- 🎨 **Custom-painted controls** — fully hand-crafted playback UI
- 🌈 **Animated gradient borders** — pulse in sync with playback via `DispatcherTimer`
- 🖥️ **SMTC Integration** — lock screen, taskbar & Bluetooth media controls
- 📁 **Auto library scanning** — discovers your local music automatically
- 🔀 **Shuffle & Repeat** — full playback queue management
- 📦 **Custom WinUI 3 Installer** — polished simulated installation experience

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| UI Framework | WinUI 3 (Windows App SDK) |
| Language | C# (.NET 8) |
| System Integration | SMTC API |
| Animations | DispatcherTimer |
| Audio | Windows Media APIs |
| Packaging | WinUI 3 Installer |

---

## 📋 Prerequisites

- Windows 10 (version 1809+) or Windows 11
- [.NET 8 Runtime](https://dotnet.microsoft.com/download/dotnet/8.0)
- [Windows App SDK](https://learn.microsoft.com/en-us/windows/apps/windows-app-sdk/downloads)
- Visual Studio 2022 with **Windows Application Development** workload
